# Beta Testing Notes

This beta build is intended for technical macOS testers.

Requirements:

- Apple Silicon Mac (arm64) — Intel Macs are not supported by this build
- Git installed and available in the shell
- any optional CLIs you personally want to use inside the embedded terminal, such as `codex` or `claude`

Important:

- this app is unsigned
- macOS may require right-click/Open the first time you launch it
- on first launch the app will ask you to select a local Git repository
- the app bundle does not include Git or external agent CLIs
